# **App Name**: GROW KISHAAN

## Core Features:

- Farmer Authentication: Secure farmer sign-up and login using phone number verification.
- Crop Listing: Farmers can upload and manage crop listings with photos, address, price, and quantity details.
- Customer Crop Browsing: Customers can browse crop listings with an intuitive interface.
- Purchase Request: Customers can request to buy crops from farmers.
- AI Listing Tool: AI-powered tool suggests listing improvements for clarity, keywords, and market appeal, based on the successful listings.
- Contact Information Sharing: Secure exchange of farmer contact details with customers upon acceptance of purchase request.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to represent freshness and nature.
- Background color: Light beige (#F5F5DC) to provide a natural, calming backdrop.
- Accent color: Soft orange (#E9967A) for call-to-action buttons and highlights to add a touch of warmth.
- Body and headline font: 'PT Sans', a humanist sans-serif that offers both a modern feel and readability.
- Use simple, line-style icons from Material-UI to maintain a clean and modern interface.
- Implement a grid-based layout to ensure responsiveness and visual consistency across devices.
